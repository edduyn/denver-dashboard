## 💰 WIP TRACKING

### ✅ Correct WIP Methodology

**Current Month WIP** = Hours on WOs **opened THIS month** that are still open at month-end.

| Scenario | WIP Impact |
|----------|------------|
| WO opened & billed **same month** | No WIP (becomes revenue directly) |
| WO opened this month, **carries to next month** | ✅ Include in WIP |
| WO open **>30 days** (from prior months) | ❌ Already WIP'd previously |

### 2026 WIP Rates
| Shop | WIP Rate | Notes |
|------|----------|-------|
| SDN | $145/hr | Standard 2026 rate |
| SDR | $145/hr | Standard 2026 rate |
| SHC | $145/hr | Standard 2026 rate |
| **SDV** | **$70/hr** | ⭐ Special WIP rate, 15% margin |

### WIP Calculation Process
```
1. Pull Anchor report (open WOs with Start Date)
2. Filter for WOs with Start Date in CURRENT MONTH
3. Cross-reference with Time Sold for hours worked
4. Calculate: Hours × WIP Rate (SDV@$70, others@$145)
5. EXCLUDE WOs opened in prior months (already WIP'd)
```

**📋 Anchor Report Naming Convention:**
- Format: `ANCHOR MM-DD HH-MM` (e.g., `ANCHOR 01-28 14-30.xls`)
- MM = Month (01-12)
- DD = Day (01-31)
- HH = Hour (00-23)
- MM = Minute (00-59)

### Auto-Calculation Logic
**A/B Ratio:**
```
Billable Hours = Sum of Hours where WOTyp IN (Component, Aircraft, Customer Assist, Billing/Sales)
Paid Hours = Total from paidhrs.xls row 1
A/B Ratio = Billable Hours / Paid Hours × 100
```

**WIP Calculation:**
```
For each WO in Anchor.xls:
  - If Shop = 'SDV': WIP = Exp Hrs × $70
  - Else: WIP = Exp Hrs × $145
  
Total WIP = Sum of all WO WIP values
```

**Employee A/B:**
```
For each employee:
  - Billable = Hours where WOTyp is billable
  - Total = All hours for employee
  - A/B% = Billable / Total × 100
```

### Processing Workflow
1. **Morning**: Upload Timesold + paidhrs → Update A/B metrics
2. **End of Day**: Upload all 5 reports → Complete update
3. Claude auto-detects Denver shops (SDN, SDV, SDR, SHC)
4. Dashboard and MD file updated together

- Denver Shops: SDN, SDV, SDR, SHC only
